# Application:
OI About

# Promo text:
Un dialog despre de sine stătător care este împărțit între aplicații.

# Description:
OI About este un dialog de sine stătător care este împărtășit între aplicații. Această aplicaţie este invocată prin meniul de alte aplicaţii OI. Acesta vă permite să vizualizaţi informaţia despre versiune, linkuri, credite, şi licența unei aplicații. Creditele includ autori, documentaţie, traducători, şi artişti. Toate aceste funcționalități sunt aduse împreună intr-un aspect simplist și eficient. 

Această aplicație nu conține reclame și nu necesită permisiunea de internet. 

Codul sursă al acestei aplicații gratuite și cu sursă deschisă este disponibil la: 
http://code.google.com/p/openintents

Pentru lista completă cu modificări și o listă cu întrebări frecvente, vizitați: 
http://www.openintents.org

Puteți ajuta la îmbunătățirea traducerii în limba dumneavoastră la Launchpad: 
https://translations.launchpad.net/openintents/trunk

Caracteristici: 
* Afişează informații generale, credite și filele cu licențe.
* Arată translatorii tuturor limbilor.
* Recuperează informație de la manifestul aplicației prin metadata și marcări implicite.
* Opțional, toate câmpurile pot fi modificare prin intermediul extra-intenţie.

Aplicații care funcționează cu „OI About”: 
* OI Notepad
* Manager OI de fișiere
* Listă de cumpărături OI
* OI Convertește CSV
* OI Safe
* OI Numărătoare inversă
* OI Lanternă

Cuvinte cheie: 
* Despre, Credite, Informații.
* OI, OpenIntents, Open Intents.
