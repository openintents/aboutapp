# Application:
OI За

# Promo text:
Самостоятелното за диалога е, че споделя между приложенията.

# Description:
OI About е самостоятелен, обяснителен диалогов прозорец, който се споделя между приложения. Това приложение се извиква през менюто на други OI приложения. То Ви позволява да преглеждате информация за версията, връзки, благодарности и лиценза на дадено приложение. Благодарностите споменават авторите, документацията, преводачите и художниците. Всичката тази функционалност е събрана заедно в опростено и ефективно оформление. 

Това приложение е безплатно и не изисква интернет за ползването ѝ. 

Изходният код на това безплатно приложение с  отворен код е достъпен на адрес: 
http://code.google.com/p/openintents

За пълен списък с промените и често задаваните въпроси,моля посетете: 
http://www.openintents.org

Можете да помогнете за подобряване на превода на вашия език в Launchpad: 
https://translations.launchpad.net/openintents/trunk

Характеристики: 
* Показва подпрозорци за обща информация, благодарности и лиценз.
* Показва преводачите на всички езици.
* Извлича информация от приложението чрез тагове по подразбиране и мета-данни.
* По желание, всички полета могат да бъдат променени чрез наличните екстри.

Приложения, които работят с "OI About": 
* OI Notepad
* OI File Manager
* Списък за пазаруване на OI
* OI Convert CSV
* OI Safe
* OI Countdown
* OI Flashlight
