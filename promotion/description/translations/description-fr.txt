# Application:
OI About

# Promo text:
Une boîte de dialogue "À propos" indépendante partagée entre applications.

# Description:
OI est d'environ un standalone sujet de dialogue qui est partagé entre les applications. Cette application est appelée par le menu des autres applications d'OI. Il vous permet de visualiser les informations de version, des liens, des crédits, et la licence d'une application. Crédits comprennent les auteurs, de documentation, de traducteurs, et les artistes. Toutes ces fonctionnalités sont réunies dans une mise en page simpliste et efficace. 

Cette application est petite annonce gratuite et ne nécessite pas l'autorisation d'Internet. 

Le code source de cette application libre et open source est disponible à l'adresse: 
http://code.google.com/p/openintents

Pour la liste complète des changements et une liste de questions fréquemment posées, s'il vous plaît visitez: 
http://www.openintents.org

Vous pouvez contribuer à améliorer la traduction dans votre langue au Launchpad: 
https://translations.launchpad.net/openintents/trunk

caractéristiques: 
* Affiche des informations générales, des crédits, et les onglets de licence.
* Affiche traducteurs de toutes les versions linguistiques.
* Récupère les informations du manifeste de l'application grâce à des balises par défaut et les métadonnées.
* En option, tous les champs peuvent être modifiés par extras intention.
