# Application:
OI Info

# Promo text:
Samostatný informační dialog "o aplikaci", který je aplikacemi sdílen.

# Description:
OI Info je samostatný dialog "O aplikaci", který je sdílen mezi aplikacemi. Tato aplikace je vyvolána přes menu jiných OI aplikací. Umožňuje Vám zobrazit si informace o verzi, odkazy, zásluhy a licenci aplikace. Zásluhy zahrnují autory, dokumentační tým, překladatele a umělce. Všechnu tuto funkcionalitu Vám přinášíme v jednoduchém a efektivním vzhledu. 

Tato aplikace je bez reklam a nevyžaduje přístup k internetu. 

Zdrojové kód této open source aplikace je dostupný na stránkách: 
http://code.google.com/p/openintents

Pro úplný seznam změn a seznam často kladených dotazů navštivte prosím: 
http://www.openintents.org

Můžete také pomoci zlepšit překlad do Vašeho jazyka na Launchpadu: 
https://translations.launchpad.net/openintents/trunk

Vlastnosti: 
* Zobrazuje záložky s obecnými informacemi, zásluhami a licencí.
* Ukazuje překladatele všech jazykových verzí.
* Získává informace z manifestu aplikace pomocí výchozích tagů a metadat.
* Volitelně mohou všechna pole být upravena pomocí extra dat v intentech.
