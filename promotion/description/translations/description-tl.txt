# Application:
OI Tungkol

# Promo text:
Isang nakapag-iisang tungkol na dyalogo na ibinabahagi sa pagitan ng mga app.

# Description:
OI Tungkol ay nakapag-iisang tungkol na dyalogo na ibinabahagi sa pagitan ng mga apps. Ang applikasyon na ito ay mapapatakbo sa pamamagitan ng menu ng iba pang mga OI na applikasyon. Ito ay nagpapahintulot sa iyo na tingnan ang bersyon impormasyon, mga link, mga kredito, at lisensya ng isang applikasyon. Ang kredito ay kasama ang mga may-akda, taga dokument, mga tagasalin, at mga artist. Lahat ng magagawa nito ay pinagsama sa simple at episyente na ayos. 

Ang applikasyon na ito ay walang patalastas at hindi nangangailangan ng pahintulot ng internet. 

Ang pinagmulang kodigo ng mga ito ay libre at bukas na pinagkukunang kodigo ng applikasyon ay magagamit sa: 
http://code.google.com/p/openintents

Para sa kumpletong listahan ng mga pagbabago at ang listahan ng mga madalas itanong, mangyaring bisitahin ang: 
http://www.openintents.org

Maaari kang makatulong upang mapabuti ang mga pagsasalin sa iyong wika sa Launchpad: 
https://translations.launchpad.net/openintents/trunk

Tampok: 
* Ipinapakita ang pangkalahatang impormasyon, mga kredito, at mga tab ng lisensya.
* Ipinapakita ang mga tagasalin ng lahat ng mga bersyon ng wika.
* Kinukuha ang impormasyon mula sa manipesto ng applikasyon sa pamamagitan ng mga default tag ng metadata.
* Opsyonal, ang lahat ng mga patlang ay maaaring mabago sa pamamagitan ng karagdagang layunin.
