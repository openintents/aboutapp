# Application:
OI About

# Promo text:
Een op zichzelf staand infovenster dat gedeeld wordt tussen apps.

# Description:
OI About is een aparte dialoog die gedeeld wordt tussen applicaties. Deze applicatie wordt gestart via het menu van andere OI-applicaties Je kunt er versie-informatie, links, credits en de licentie van een applicatie mee bekijken. In de credits staan auteurs, documentatie, vertalers en artiesten. Al deze functies komen samen in een simpele en efficiënte lay-out. 

Deze applicatie is advertentievrij, en heeft geen internet toegang nodig. 

De broncode van deze gratis en open source applicatie is beschikbaar op: 
http://code.google.com/p/openintents

Het complete overzicht van wijzigingen en de FAQ is te lezen op: 
http://www.openintents.org

Jij kan helpen om de vertaling in je eigen taal te verbeteren via Launchpad: 
https://translations.launchpad.net/openintents/trunk

Eigenschappen: 
* Toont algemene info, credits en licentie-tabs.
* Toont vertalers van alle taalversies.
* Haalt info op van een applicatie-manifest, via standaard tags en metadata.
* Alle velden kunnen eventueel worden aangepast via Intent Extra's.

Applicaties die werken met OI About: 
* OI Notepad
* OI Bestandsbeheer
* OI Boodschappenlijstje
* OI Convert CSV
* OI Kluis
* OI Countdown
* OI Zaklamp
