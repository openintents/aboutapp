# Application:
OI About

# Promo text:
Uma aplicação de informações transversais das aplicações.

# Description:
O OI About é uma aplicação de informações transversais das aplicações. Esta aplicação é invocada através do menu das outras aplicações OI. Permite-lhe ver as informações da versão, ligações, créditos e a licença. Os créditos incluem os autores, documentação, tradutores e artistas. Todas as funcionalidades são disponibilizadas num esquema simples e fácil de utilizar. 

Esta aplicação é livre e não requer permissões de Internet. 

O código fonte desta aplicação está disponível em: 
http://code.google.com/p/openintents

Para ver a lista de alterações e as perguntas frequentes, consulte: 
http://www.openintents.org

Pode ajudar a traduzir as aplicações no Launchpad: 
https://translations.launchpad.net/openintents/trunk

Características: 
* Mostra as informações, os créditos e a licença.
* Mostra os tradutores das aplicações.
* Recolhe as informações das aplicações através de "tags" e metadados.
* Opcionalmente, todos os campos podem ser modificados.

As aplicações vinculadas ao OI About: 
* OI Notepad
* OI File manager
* OI Shopping List
* OI Convert CSV
* OI Safe
* OI Countdown
* OI Flashlight
