# Application:
OI About

# Promo text:
Un diálogo «Acerca de» autónomo que se comparte entre aplicaciones.

# Description:
"About OI" es un diálogo independiente que es compartido entre aplicaciones. Esta aplicación se ejecuta a través del menú de otras aplicaciones OI. Le permite ver información de la versión, enlaces, créditos, y la licencia de una aplicación. Los créditos incluyen autores, documentación, traductores, y artistas. Todas estas funciones se organizan en un diseño simple y eficiente. 

Esta aplicación no tiene anuncios y no requiere permisos de internet. 

El código fuente de esta aplicación libre y de código abierto está disponible en: 
http://code.google.com/p/openintents

Para la lista completa de cambios y la lista de preguntas frecuentes, por favor visite: 
http://www.openintents.org

Usted puede ayudar a mejorar la traducción de su idioma en Launchpad: 
https://translations.launchpad.net/openintents/trunk

Características: 
* Muestra información general, créditos y las pestañas de licencia.
* Muestra los traductores de todas las versiones localizadas
* Obtiene información del manifiesto de la aplicación a través de las etiquetas por defecto y los metadatos.
* Opcionalmente, todos los campos pueden ser modificados a través de otras acciones.

Aplicaciones que funcionan con "OI About": 
* Bloc de notas OI
* Gestor de archivos OI
* Lista de compras OI
* OI Convert CSV
* OI Safe
* OI Countdown
* OI Flashlight
